# SaaS Boilerplate

## Command

- server: `npm run dev`
- client: `npm start`

## Tech Stack

- Frontend: ReactJS
- Backend: ExpressJS
- Database: MongoDB
- Mail: Nodemailer

## Features

- Email and password registration and login
- Google sign up and login
- JWT based authentication
- Role based authorization
- Confirmation emails
- Password reset through email
- Account activation reverification email
- JSON APIs
